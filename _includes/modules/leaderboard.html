Leaderboard

The purpose of this module is to present data (typically statistics) within a quick rollup area. This is mainly used within statistic reporting views.

Layout Variations

While all current instances of this module use the 4-column layout, other layout variations are available as well. The class name structure is similar to the grid, but different in its implementation. Instead of adding an extra class (.-col[n]) onto every div.-item element, state the desired element column width size within the leaderboard wrapper.
