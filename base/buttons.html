Warning: Be careful when marking up links with the button role. Buttons are expected to be triggered using the Space key, while links are expected to be triggered using the Enter key. In other words, when links are used to behave like buttons, adding role="button" alone is not sufficient. Like a secondary button in purpose, this stylistic variation is to be used when you desire an even simpler button style—especially when used in concert with the primary button style. This style works well within small UI actions areas that are tertiary in importance or when a larger button size would draw too much attention to the button.

Clear
