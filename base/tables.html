Tables

The base style has minimal styling applied to it. It establishes preferred padding, font-size, and font-weight treatments. To enhance the look of the table, apply any additional classes listed below within the Modifiers section.

Modifiers

Borders

When tables have multiple columns, it is suggested to add row borders to help the user more easily track the table data. This is suggested for data tables such as found within reports.

Hover states

Condensed

The condensed table does pretty much what it sounds like it would do: condenses the padding and line-height so that the table has a tighter feeling to it. This is suggested for data tables such as found within reports that have multiple records.
