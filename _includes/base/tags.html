Base Style

Tags are a core feature on Stack Overflow. These tags help users quickly identify question subject matter. The use of tags on Stack Overflow Careers however is somewhat similar. Tags on Careers are used to help users quickly identify the type of job. They are also used (currently) to help companies list their tech stack.

Lists of Tags

Using .postTag on individual tags can be cumbersome if you have a list of tags. You can also apply .postTags to an unordered list to make all the list items styled as tags on the same line.
