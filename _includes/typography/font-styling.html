Headlines

Headlines provide signposts for content as well express content priority. Headlines should descend in order. This means that a h3 shouldn't precede a h1 in the DOM order. The question to ask yourself is, "If this page's CSS didn't load, would this copy hierachy still make sense?"

Paragraphs

The top design element that we use is written language. Our job is to divide, organize, and interpret this written language so that users can achieve their goals quickly. This is achieved through optmizing typography for "readability, accessibility, usability, and overall ... balance."
