Badges

Base Style

The base badge style seeks to establish the general sizing and layout. Box padding, font-size, and font-weight treatments are all set here. Badge color combinations are listd within the badge styles section.

The default badge style should not be used independently of a badge color combination. Badges are used for specific purposes and should not be re-purposed for different areas.

Badge Styles

Badge styles are created for specific instances. In the interest of not confusing users, color combinations should only be used for the specific instances listed below. Using a style for multiple items can create confusion for users. While the labels might be different, we cannot assume people read closely. Color block are added as a foraging aid to help users quickly identify items while scanning instead of reading every word on a page. If you need a unique styles, please consult the Careers product design team.
