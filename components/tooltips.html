CSS only tooltip

The easiest way to display a tooltip is to use the css only version. The tooltip will automatically takes 100% of its parent's width. You can set a defined size of 200px using .size-200.

            
                Trigger tooltip
                
                    I'm a CSS only tooltip!
                
            

            
                200px width tooltip
                
                    I'm a 200px wide CSS only tooltip!
Trigger tooltip I'm a CSS only tooltip! 200px width tooltip I'm a 200px wide CSS only tooltip!
